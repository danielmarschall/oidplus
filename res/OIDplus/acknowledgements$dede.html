Informationen über OIDplus

OIDplus ist eine OpenSource-Datenbank von ViaThinkSoft , mit der Registrierungsstellen ihre Objekte wie Objektkennungen (OIDs) oder GUIDs (Global Unique Identifiers) und vieles mehr verwalten können.

Betreiben Sie Ihre eigene Registrierungsstelle

Um Ihre eigene Registrierungsstelle zu betreiben, können Sie OIDplus auf Ihrem eigenen Server installieren. Es ist kostenlos!

Klicken Sie hier um weitere Informationen zu erhalten

Danksagung

ViaThinkSoft OIDplus verwendet die folgenden Produkte von Drittanbietern:

  • 0xbb SHA3-Implementierung in PHP (MIT)
  • bcrypt.js von dcode.io (New-BSD / MIT-Lizenz)
  • bignumber.js (MIT)
  • Bootstrap (MIT)
  • CSS / JS Minify von Matthias Mullie (MIT)
  • emn178 SHA3-Implementierung in JavaScript (MIT)
  • FileFormats von ViaThinkSoft (Apache 2.0-Lizenz)
  • Firebase PHP-JWT (BSD-3-Clause)
  • GedMarc layout (MIT / GPLv3), welches ein Fork von allpro layout (MIT / GPLv3) ist
  • jQuery (MIT)
  • jQuery UI (MIT)
  • jsTree (MIT)
  • MBString Polyfill von Symfony (MIT-Lizenz)
  • Nuvola Icon Pack von David Vignoni (LGPL)
  • PathConverter von Matthias Mullie (MIT)
  • PHP HTTP Client von Manuel Lemos (BSD Lizenz)
  • PHP SVN Client von Sixdegrees / Cesar D. Rodas (Neue BSD-Lizenz)
  • Polyfill v3 (MIT)
  • ReCaptcha (BSD 3-Klausel "Neue" oder "Überarbeitete" Lizenz)
  • SpamSpan (GPLv2)
  • TinyMCE (LGPL 2.1)
  • VNag von ViaThinkSoft (Apache 2.0 Lizenz)
